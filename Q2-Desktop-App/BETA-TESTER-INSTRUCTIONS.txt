## 📋 System Requirements

## 🚀 Quick Start (2 minutes)

### Installation:
1. **Extract** this ZIP file to any folder (e.g., Desktop)
2. **Navigate** to the extracted folder
3. **Double-click** "Q2 Desktop.exe" to launch
4. **If prompted**, allow the app to run (it's safe!)

### First Test:
1. **Stage 1**: Generate a prompt about your subject area
2. **Stage 2**: Use the prompt with ChatGPT/Copilot, paste response
3. **Stage 3**: Run validation to see your results

## 🆘 Troubleshooting

### Common Issues:
- **App won't start**: Run as Administrator
- **Python not found**: Install Python from python.org
- **Validation fails**: Check file permissions in app folder
- **LLM errors**: Verify your JSON format in Stage 2

### Getting Help:
- Check console messages (Ctrl+Shift+I)
- Try with different question types
- Test with smaller question sets first
- Restart the app if it becomes unresponsive
